<h1 align="center">Wirecard Bundle</h1>
<p align="center">:package: :credit_card: Bundle for symfony with wirecard payment gateway</p>

## Setting

Configuration your .env


**WIRECARD_TOKEN**=YOUR TOKEN
<br>
**WIRECARD_KEY**=YOUR KEY
